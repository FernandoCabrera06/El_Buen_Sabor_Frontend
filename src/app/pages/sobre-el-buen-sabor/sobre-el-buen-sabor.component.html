Proyecto Final "El Buen Sabor"

El delivery de comidas de la ciudad "El Buen Sabor", ofrece a sus clientes una amplia variedad de comidas como sandwiches de lomo, hamburguesas, papas fritas y empanadas, ademas de bebidas para acompañar, con surtido en variedades de gaseosas, vinos y cervezas. La app posee un catálogo donde los clientes encuentran a su disposición todas las comidas disponibles, pudiendo ver composición, precio y agregarlas al carrito para realizar el pedido. Para poder realizar compras es necesario registrarse con correo electrónico y una clave de acceso y comenzar a navefgar por la app para realizar los pedidos deseados.
Este proyecto fue planteado como Proyecto Final para Laboratorio de Computación IV, y el mismo está creado con una arquitectura cliente-servidor. El backend es una API RestFul realizada en Java 8 con SpringBoot, la información es almacenada en una base de datos relacional MySQL y las interfaces gráficas de usuario se realizaron con Angular 13 y TypeScript para lograr que la app sea resposive. Además, para la pasarela de pago se utilizó la API de Mercado Pago, permitiendo a los clientes realizar los pagos con cualquier tarjeta de crédito o débito.
